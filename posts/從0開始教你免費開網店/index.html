從0開始教你免費開網店

想擁有一間屬於自己的網店，需要花很多錢嗎？NO！

這是免費的！FREE！

甚麼是「屬於自己」的網店？

在這裡屬於自己的意思是是從伺服器到網店的源碼所有東西都是你可以控制的，不像 Shopify、Shopline 這類綁定平台以及需要付月費才能解鎖功能的網店。以前我們或許要支付伺服器的租用費，或是自架伺服器時所需的電費，現在我們通通可以免費實現！而且他日你需要找網設計師（例如小弟）的時候他也會很感激你喔！

這裡唯一需要用到錢的地方大概就是購買網域吧，網域的部份我會在另一篇文章再做說明

在這裡你可以學到…

這篇文章會帶領你一步步建立一間網店，當然你可能需要一點點電腦知識，就一點點而且，如果有問題，也可以電郵給我，我很樂意解答你的問題！

準備好了嗎？好，以下是你需要的東西，先看看就好，之後會逐一解說的

  • Google 免費伺服器
  • 信用卡 (註冊用)
  • Wordpress
  • WooCommerce （Wordpress 插件）

註冊 Google Cloud Platform

Google Cloud Platform 簡稱 GCP 是 Google 的雲端服務，Google 在 2019 年推出了一個「always free」免費計劃，當中包括了這項 30GB 空間的 Compute Engine，這就意味著我們伺服器只要不超過配置就是完全免費的！

以下是註冊網址，大家都是聰明人，就不詳細說明了

https://cloud.google.com/

建立伺服器

註冊後到 Google Console 建立一個新項目，在網頁上邊找到 Select a project，然後再按 NEW PROJECT

這邊輸入項目名稱，我就叫它 Online Shop 好了

之後就到建立伺服器的部份，從 Sidebar 找到 Compute Engine 的 VM instances

在 Marketplace 搜尋這個 Google 團隊製作的 LAMP Stack

先不要理會那個估算的價錢，根據我的經驗，只要符合 Goole 當時的免費配置，在月結的時候會以 Discount 形式扣減費用，所以這邊先按下 Launch

因為這次的主題是免費，我們只能使用唯一免費的 f1-micro，值得注意的是 Google 在這段時間建立新帳號就立即送 $300 USD credit，所以即使選了比較高階的伺服器也有足夠的免費金額讓你試用而不會立即從你的信用卡扣帳，不過這筆錢是有期限的，所以你可以把它花光然後才降級也是 OK 的

同樣地我們可以免費使用最多 30GB 的 HDD 硬碟，所以我就把它先調到 30GB，然後我們把 Allow HTTPS traffic from internet 打勾，以後在域名方面一定會用到的

最後按下 Deploy，運行结束後注意右邊的選單，我們需要記下這些重要的資料，之後會用到

Site address

網店的 IP 地址，在擁有個人域名之前，我們可以使用 IP 瀏覽網站

MySQL user / password

資料庫的用戶名稱和密碼

也可以之後再搜尋 Deployment Manager 回到這個頁面

安裝 Wordpress

Wordpress 是市面其中一個最多元化的內容管理系統，在整個安裝過程我們都只會使用指令！但是不用擔心，因為很簡單，過程只要 10 分鐘就好了，首先我們回到 VM Instance 的頁面

在你的虛擬伺服器右邊找到 SSH 按鈕並按下去，然後就會彈出這個像黑客使用的 SSH 頁面，是不是很 COOL？

以下我已經給你準備好指令，只要用 Ctrl + C 複製，再在頁面上貼上 Ctrl + V，最後按 Enter

謎之聲：你當我白痴啊？

Copy Code

sudo apt install unzip -y &&
cd ~/ &&
wget https://zh-hk.wordpress.org/latest-zh_HK.zip &&
unzip latest-zh_HK.zip -d /var/www/html/ &&
cd /var/www/html &&
sudo chown -R www-data /var/www/html/wordpress/

等它跑完以後使用 ls 指令檢查一下有沒有 wordpress 這個資料夾

建立資料庫

另外我們需要一個資料庫給 Wordpress 使用，還記得之前叫你記下來的 MySQL user / password 嗎？這邊我們就要用這些資料登入 MySQL 然後建立一個資料庫，輸入指令

Copy Code

mysql -u root -p

然後輸入密碼，密碼不會在頁面顯示出來，我們可以用 Ctrl + V 貼上，然後按 Enter

登入後會看到指令輸入位置的前端變成>，執行以下指令建立名為 wordpress 的資料庫，然後第二句指令確認是否建立成功

Copy Code

create database wordpress;
show databases;

最後使用以下指令登出 MySQL

Copy Code

exit

設定 Wordpress

來到這裡最困難的部份已經完成囉！還記得之前的 IP 地址嗎？我們現在就會使用 IP 進入網站然後再設定 Wordpress，以我的 IP 為例就是

http://34.94.31.255/wordpress

把剛建立的資料庫資料填上，密碼不要跟著我填喔～

如無意外，會出現這樣的頁面

如果你出現了這句「抱歉，但我無法寫入 wp-config.php 檔案。」的錯誤，就表示在你伺服器的 wordpress 資料夾權限沒有設定好，你可以再運行一次這句指令

Copy Code

sudo chown -R www-data /var/www/html/wordpress/

這是很常見的錯誤喔～

接下來輸入網站標題（之後可以更改）和設定管理員的帳戶和密碼

大功告成！

安裝 WooCommerce

登入後我們會進入管理台，這是只有管理員才會看到的頁面，再次瀏覽你的網頁會發現一些類似部落格的內容，因為 Wordpress 本來就是讓人寫網誌的

讓它變成網店的關鍵就是一個叫 WooCommerce 的免費插件（外掛），在安裝外掛的頁面搜尋 WooCommerce，接著安裝並啟用這個插件

跟著指示進行一系列的設定

這個部份必須勾選 Storefront 佈景主題，套用它的主題會令你的網站從部落格變成網店的樣子

新增商品

設定好了以後，我會可以從左邊的選單新增第一件商品

填上商品的資料，然後按左上的發表

再次回到自己的網店主頁，就可以看到剛建立的商品了！

http://34.94.31.255/wordpress

客製化你的網店

為了讓我們的網站更像一間網店，我們可以客製化 Storefront 佈景主題，在外觀選單找到佈景主題，再自訂使用中的 Storefront 主題

在這裡我們可以修改網站標題等等的資料，這邊我會把首頁設定成商店

把無關的小工具逐個刪除

刪除後整個 Sidebar 都消失了！

最後我們把目錄的範例頁面也刪掉就完成了！
